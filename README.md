# Soccer Simulator

A text-based soccer simulation game with league standings, team details, player profiles, and weekly schedules.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
3. Open your browser and navigate to `http://localhost:3000`

Deployed game available at: https://anthonymarzotto.github.io/soccer_sim_game/
