ChangeLog

Changes by version

v0.5.0

2026-05-04

This release implements the Player Progression system, enabling dynamic growth or decline of player attributes during season rollover based on age phase and potential. The Player Profile view has been enhanced to display prior year attribute comparisons (+/- deltas) and an Attributes History table. The data schema version was bumped to H to persist this new progression metadata; existing leagues must be regenerated.

v0.4.0

2026-04-29

This release introduces a comprehensive player injury system integrated directly into the match simulation. Players can now sustain injuries during matches that sideline them for multiple games, requiring deeper squad rotation and forcing CPU teams to actively utilize their bench reserves. The team details and roster views have also been updated to clearly display unavailable players in a dedicated section.

v0.3.3

2026-04-24

Fixed player match rating formula with a fixed base of 50, persisted per-season average ratings and star nominations to career stats, and added Stars of the Game display on match summary. Live ratings now appear in watch mode alongside player names, and tackle/interception attribution was improved to correctly credit the defensive winner based on proximity and attribute weighting.

v0.3.2

2026-04-23

This release expands the player attribute model with nine new stats — three visible goalkeeping attributes (Handling, Reflexes, Command of Area) and six hidden mental/physical stats — replacing the single Goalkeeping stat with a dedicated Goalkeeping section on the player profile. Player objects now track mood and fatigue as persistent fields initialized to 100 each season. The data schema version was bumped to E; existing leagues must be regenerated.

v0.3.1

2026-04-21

This release introduces dual team-color support across match visuals and team badges to improve contrast and readability when clubs have similar primary colors. Watch mode now uses distinct main and accent colors for formation dots and roster-card edge highlights, and badge rendering was refined so gradient badges fade from main to accent while shield and jersey icons preserve clean transparent styling. The team color service now provides stable primary/secondary palette values for consistent styling across components.

v0.3.0

2026-04-17

This release adds a dedicated player stats page with season-by-season and career views, plus richer player profile and team detail links into those records. It also extends league metadata and persistence with the current season year so generated career stats, saved data, and season rollups stay aligned across reloads.

v0.2.0

2026-04-13

Watch mode now uses simulation-authored formation snapshots and fatigue timelines so substitutions, shape shifts, and player stamina stay aligned with generated match outcomes. Commentary replay gained a live scrollable log with auto-follow, expand/collapse, adjustable playback speed, and pause/resume controls to let players experience matches at their own pace. Testing coverage was extended with targeted unit tests for commentary state transitions, scroll threshold behaviour, and auto-follow toggling, and DOM access was refactored to Angular @ViewChild conventions.

v0.1.0

2026-04-08

This release moves match simulation to the new Variant B engine with adaptive tick flow, stronger possession logic, and shape-aware manpower behavior after dismissals. It also expands realism with foul/card, carry, and pass resolution updates, including direct shape-aware pass success and failure handling. Guardrail coverage was widened with dedicated calibration, manpower, substitutions, and A/B scenario tests, and the simulation flow documentation was updated to match.
