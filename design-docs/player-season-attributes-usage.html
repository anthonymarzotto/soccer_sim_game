Read Path

Each attribute travels through the same four stages. Most never make it past stage 2.

1. Generate

GeneratorService.generatePlayer rolls every stat via randomStat(min, max, teamQuality) with position boosts for forwards, midfielders, defenders, and goalkeepers.

2. Bake overall

overall is computed once, immediately after generation, via two asymmetric formulas:

outfield: floor( speed+strength+flair+vision+determination
+tackling+shooting+heading+longPassing+shortPassing ) ÷ 10 goalkeeper: same 10 + handling×2 + reflexes×2 + commandOfArea ) ÷ 15

Never recomputed — including at rollover. The goalkeeper divisor compensates for the three extra weighted GK terms; if you add a stat to the GK sum, update the denominator too.

3. Resolve season

getCurrentPlayerSeasonAttributes(player, seasonYear) returns the active record. A missing record is treated as a persistence integrity failure and throws.

4. Consume

FieldService derives team tactics; MatchSimulationVariantBService reads stats inside action resolution; GameService uses overall as a quality proxy; StatisticsService seeds player ratings from overall.

Principles to remember

Stat values never change mid-match

Variant B always reads from the active season record. In-match dynamics live in transient state — PlayerFatigue (currentStamina / fatigueLevel / performanceModifier, initialised fresh each match and never persisted), possession chain, scoreline, and RNG — none of which writes back into PlayerSeasonAttributes.

overall is a snapshot, not a derivative

Generated once from a fixed subset and never recomputed. Edits to source stats after generation will silently desync from overall, and most quality-driven UI/logic keys off overall.

Team tactics use averages, not stars

FieldService.calculateTeamTactics averages overall, speed, shortPassing, longPassing, tackling, and shooting across the whole roster. One elite player will not flip the team's playing style.

Rollover is a copy, not a progression

GameService.startNewSeason deep-copies every stat object into the next season and resets mood/fatigue. There is no automatic aging, training, or decay step.
