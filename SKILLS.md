# Development Skills

This directory contains reusable workflows and development patterns for the soccer simulation project.

## Available Skills

### [Add New Tuning Variables to Variant B Engine](./.agents/skills/add-tuning-variables/SKILL.md)
Step-by-step workflow for extending the Variant B simulation engine with new tunable parameters. Covers engine type definitions, debug UI integration, and acceptance guardrail testing.

### [Version And ChangeLog Update](./.agents/skills/version-bump/SKILL.md)
Update app version metadata and the ChangeLog page for a branch. Use when you need to increment package.json version, increment dataSchemaVersion, sync generated schema version file, and add a 1-3 sentence summary of branch changes.
